Welcome to the ExtinctAPI

This API provides data about extinct animals and birds. You can use it to get information like species name, extinction year, and location.

API Documentation

The ExtinctAPI allows you to retrieve information about various extinct species.

Base URL

https://extinct-api-1.onrender.com

Endpoints

GET /animals

Fetches a list of extinct animals with their details.

Example Request:
GET https://extinct-api-1.onrender.com/animals
Example Response:
[
  {
    "name": "Dodo",
    "species": "Raphus cucullatus",
    "extinct_year": 1662,
    "location": "Mauritius"
  },
  {
    "name": "Passenger Pigeon",
    "species": "Ectopistes migratorius",
    "extinct_year": 1914,
    "location": "North America"
  }
]

How to Use the API

You can make HTTP GET requests to the endpoint to retrieve data.

Python Example:


import requests

response = requests.get('https://extinct-api-1.onrender.com/animals')
data = response.json()
print(data)

JavaScript Example (for web):


fetch('https://extinct-api-1.onrender.com/animals')
  .then(response => response.json())
  .then(data => console.log(data));
